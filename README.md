# Foodchemy

Foodchemy 是一個用純前端製作的 emoji 食物合成遊戲，靈感來自 Little Alchemy。玩家把食材拖進合成區，讓材料重合後觸發配方，逐步解鎖更多食材；部分最終成果則會被記錄為「食譜」，而不會回到食材面板。

## 專案檔案

- `index.html`：頁面結構與右側控制面板
- `styles.css`：整體介面樣式、彈窗與圖鑑外觀
- `app.js`：拖曳、碰撞、合成、解鎖、圖鑑邏輯
- `main.py`：早期 `tkinter` 原型，現在主要版本不是它

## 目前玩法

- 初始基本元素：`🌱 🧬 🎲 🔪 💧 🔥 ⏳ 🌍`
- 左側已解鎖元素可以：
  - 點一下快速生成到合成區
  - 直接拖進合成區
- 合成區中的元素可以：
  - 左鍵拖曳
  - 右鍵單獨刪除
- 合成成功時：
  - 若結果是 emoji，會生成到場上並加入已解鎖食材
  - 若結果是文字名稱，會視為食譜，只記錄在食譜合成樹，不會回到食材面板或場上

## 圖鑑與模式

- `開發者模式`
  - 只影響食材面板的顯示，勾選後可暫時使用所有食材
  - 不會直接解鎖食譜
- `食材合成樹`
  - 顯示所有可合成出 emoji 食材的配方
- `食譜合成樹`
  - 顯示所有合成結果為文字名稱的配方
  - 已實際合成出的食譜會以綠底標示

## 已支援的特殊規則

- 隨機配方會依結果平均分配機率，例如：
  - `🌱 + 🎲 -> 🥔 / 🌽 / 🌾`
  - `🧬 + 🎲 -> 🐂 / 🐓 / 🐖 / 🦆 / 🦠`
  - `🪴 + 🎲 -> 🍅 / 🌶️`
- 食譜結果目前包含：
  - `🍜 + 🥩 = 牛肉麵`

## 開啟方式

最簡單的方法是直接用瀏覽器打開 `index.html`。

如果你想用本機伺服器開啟，也可以在專案資料夾執行：

```powershell
py -m http.server 8000
```

再到瀏覽器開啟：

```text
http://localhost:8000
```

## 操作方式

1. 從左側已解鎖元素面板拿出食材。
2. 把食材拖到中央合成區。
3. 讓配方材料重合後放開滑鼠。
4. 觀察右側訊息欄與解鎖進度。
5. 需要整理場面時，可用右鍵刪單一元素或按 `重置場上元素`。

## 備註

- 目前正式版本是網頁版，主要入口是 `index.html`。
- 瀏覽器版的好處是彩色 emoji 顯示比桌面 `tkinter` 方案穩定。
